ws28

ws28 — Communicate with WS2803/WS2812

Synopsis

void                ws28_init                           (void);
void                ws28_send_color                     (const color_t *src);
void                ws28_send                           (fb_t *fb);

Description

Arduino-compatible functions for sending color_t and fb_t to a WS2803 or WS2812 LED controller chip.

Warning

WS2811 chips (and perhaps others) use a different signaling protocol. This library will not work with those.

Details

ws28_init ()

void                ws28_init                           (void);

Initialize the SPI interface for use with a WS2803 or WS2812 (not WS2811)

ws28_send_color ()

void                ws28_send_color                     (const color_t *src);

Send Red, Green, and Blue values to the WS28xx, in that order, not pausing for a reset.

See also: ws28_send(), SPI.transfer()

src :

 Color to send. [in]

ws28_send ()

void                ws28_send                           (fb_t *fb);

Send an entire fb_t to the WS28xx, pausing before send if necessary to reset the WS28xx's to the first pixel.

The pixels are sent in column-major order; i.e., all the pixels with x==0 are sent, then all with x==1, etc.

fb :

 Framebuffer to send. [in]
