font

font — Basic font drawing support.

Description

Contains functions for drawing characters and strings to a fb_t.

Note

These function draw inside the framebuffer only. They will skip drawing locations outside the framebuffer, so you can use them to scroll text off-screen.

Note

The background is not drawn by these functions.
